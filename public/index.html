Front Page

Basic online shop web page done as a project for Web Development 1 course in Tampere University.

Utilizes Node.js Server-Client communication.

Implements basic CRUD operations using MongoDB document model database, that is used for the users and products.

User passwords are encrypted, and saved into the database as hashed values.
